# Apache Spark Assignment

## Overview

This repository contains solutions to an Apache Spark assignment implemented using **PySpark** in **Google Colab**. The assignment covers Spark architecture, DataFrame operations, transformations, actions, CSV and Parquet file handling, filtering, column manipulation, lazy evaluation, DAG, and fault tolerance with practical examples.

---

## Technologies Used

- Python
- Apache Spark (PySpark)
- Google Colab
- Pandas

---

## Assignment Topics

### Theory
- Spark Architecture
- Driver, Cluster Manager, and Executor
- Lazy Evaluation
- DAG (Lineage Graph)
- CSV vs Parquet
- Predicate Pushdown
- Transformations vs Actions
- Client Mode vs Cluster Mode
- show() vs collect()

### Practical
- Reading CSV files
- Reading and writing Parquet files
- DataFrame filtering
- Selecting columns
- Renaming columns
- Casting data types
- Creating new columns
- Saving DataFrames as CSV
- Handling null values
